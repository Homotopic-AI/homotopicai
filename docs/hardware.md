    <div class="performance-metrics mb-5">
        <h3>Performance Metrics & Measurement Protocol</h3>
        
        <h4>Core Metrics</h4>
        <ul class="feature-list">
            <li><i class="fas fa-check"></i> <strong>FLOPS (Floating-Point Operations per Second):</strong> Measure peak performance for dense matrix multiplication (e.g., FP32/FP64)</li>
            <li><i class="fas fa-check"></i> <strong>TOPS (Tera Operations per Second):</strong> For integer-based tasks (e.g., AI inference, cryptography)</li>
            <li><i class="fas fa-check"></i> <strong>Mathematical Primitives:</strong> Benchmark domain-specific operations including:
                <ul>
                    <li>Matrix Inversion: Time to solve A⁻¹ for large matrices (e.g., 10k × 10k)</li>
                    <li>PDE Solving: Time to resolve partial differential equations (e.g., Navier-Stokes)</li>
                    <li>Cryptographic Throughput: Lattice-based encryption/decryption speed (e.g., Kyber-1024)</li>
                </ul>
            </li>
        </ul>

        <h4>Test Conditions</h4>
        <ul class="feature-list">
            <li><i class="fas fa-check"></i> <strong>MAP Devices:</strong>
                <ul>
                    <li>Portable, battery-powered units (e.g., smartphone-sized)</li>
                    <li>Ambient temperature (20-25°C), no active cooling beyond passive heat sink</li>
                </ul>
            </li>
            <li><i class="fas fa-check"></i> <strong>Data Centers:</strong>

            </li>
        </ul>
